# 数据集快速说明

> 课程提供的 `liver.h5ad`（约 1.4 GB，~7 万细胞 × 30 维 PCA）字段速览与读取片段，便于二次开发者快速理解数据结构。完整数据流转与预处理见 [`05_用户手册.md`](05_用户手册.md)。

## 数据导入
```python
import scanpy as sc

adata = sc.read_h5ad("data/raw/liver.h5ad")
print(adata)
print("细胞数:", adata.n_obs)
print("基因数:", adata.n_vars)
```

## 数据字段说明

数据采用 `.h5ad` 格式存储

- `obs`：细胞（cell）层面的元数据
- `var`：基因（gene）层面的注释    
- `obsm`：降维结果（PCA/UMAP/tSNE）   
- `uns`：全局数据描述信息

PCA 降维结果保存在 `obsm.X_pca` 字段：

```python
X_pca = adata.obsm["X_pca"]
print(X_pca.shape)
```

## 可视化

print(adata.obs["cell_type"].value_counts())
